# sistema-gestion-pacientes-ccss
Proyecto final de Bases de Datos NoSQL - Sistema de Gestión de Pacientes para la CCSS
## Tecnologías

- MongoDB
- Node.js
- Express.js
- Mongoose

## Descripción

Sistema diseñado para gestionar pacientes, consultas médicas, exámenes de laboratorio e imágenes médicas utilizando una base de datos NoSQL orientada a documentos.
